# Action Pack for Azure

> Note: The use of the scripts requires the PowerShell Module Az.

+ [Get-AzDatabases.ps1](./Get-AzDatabases.ps1)

    Gets one or more databases

+ [Get-AzElasticPoolActivities.ps1](./Get-AzElasticPoolActivities.ps1)

    Gets the status of operations on an elastic pool

+ [Get-AzElasticPoolDatabases.ps1](./Get-AzElasticPoolDatabases.ps1)

    Gets elastic databases in an elastic pool and their property values

+ [Get-AzElasticPools.ps1](./Get-AzElasticPools.ps1)

    Gets elastic pools and their property values in an Azure SQL Database

+ [New-AzDatabase.ps1](./New-AzDatabase.ps1)

    Creates a database or an elastic database

+ [New-AzElasticPool.ps1](./New-AzElasticPool.ps1)

    Creates an elastic database pool for a SQL Database

+ [Remove-AzDatabase.ps1](./Remove-AzDatabase.ps1)

    Removes an Azure SQL database

+ [Remove-AzElasticPool.ps1](./Remove-AzElasticPool.ps1)

    Deletes an elastic database pool

+ [Rename-AzDatabase.ps1](./Rename-AzDatabase.ps1)

    Renames a database or an elastic database

+ [Set-AzDatabase.ps1](./Set-AzDatabase.ps1)

    Sets properties for a database, or moves an existing database into an elastic pool

+ [Set-AzElasticPool.ps1](./Set-AzElasticPool.ps1)

    Modifies properties of an elastic database pool in Azure SQL Database
